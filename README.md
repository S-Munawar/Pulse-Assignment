# Pulse Assignment

A features management dashboard built with Next.js, React, and Tailwind CSS.

## Features

- Hierarchical tree view of features and folders
- Search and filter functionality
- Customizable column visibility
- Detail panel for feature information

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
